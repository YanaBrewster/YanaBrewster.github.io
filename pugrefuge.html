Pug Refuge

Wordpress CMS Theme

The Yoobee assignment was to create a custom theme on Wordpress with a mix of custom and third party functionality. The client needed to be able to update the content on every page of the site easily. The site had to be secure, integrate SEO and be optimised for performance and compatibility.

Design

Initial sketches for desktop and mobile were drawn up to begin the design process.

User Testing

Usability Testing (medium and high fidelity prototypes) was completed on UXPin on desktop and mobile. Beta Testing and Browser Testing was also conducted on the live site.

Front-end Development

A use case UML diagram was create for the user and client, outlining the delieverables of the project.
